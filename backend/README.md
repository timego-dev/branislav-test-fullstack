## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start
```

## Migration
Please run migration sync first before running the app. 
```bash
# development
$ npm run migrate
```

## Test Before migration
Please set NODE_ENV value as "test" first.
```bash
# development
$ npm run test-migrate
```

## Test
```bash
# development
$ npm run test
```
